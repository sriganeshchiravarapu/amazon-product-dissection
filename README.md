Amazon Product Dissection

This repository contains a detailed product dissection case study and relational database schema design for Amazon. The project was created as part of the AlmaBetter Data Science and AI curriculum.

## Project Overview

Amazon is one of the world’s largest e-commerce platforms, connecting millions of users with products offered by sellers worldwide. This project investigates the real-world challenges addressed by Amazon and presents a relational database schema that supports its core business functionalities.

## Repository Contents

- **Product Dissection Case Study**: A comprehensive analysis of Amazon’s platform features and the challenges it addresses in real-world e-commerce scenarios.

- **Schema Description**: An overview of the database entities and their attributes that model the platform. 

- **ER Diagram**: A visual representation of the relationships between entities in the relational database design.

## Entities and Attributes

The relational schema includes the following entities:

- **User**
- **Product**
- **Seller**
- **Order**
- **Review**

Detailed descriptions of each entity and their respective attributes are available in the main project document.

## ER Diagram

The ER diagram illustrates the relationships between users, products, sellers, orders, and reviews within the Amazon platform. It provides a clear understanding of how data flows across the system to support large-scale operations. The ER diagram image is included in this repository.

## How to Use

This project is an academic case study and does not require installation or execution steps. Review the document and ER diagram to understand the database structure and design rationale.
